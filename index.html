Policy Exhaustive Mode

FBA 费成本批量测算器

导入一份 SKU 包装后单件商品尺寸重量表，直接穷举美国站 FBA 所有政策可能场景：标准 FBA、低价 FBA、标准尺寸 SIPP 折扣、大件非 SIPP 包装费、燃油物流附加费、低量库存费各档，并标注每种场景成立条件。

当前 HTML 版为纯本地运行，不依赖 Python。支持在线逐行录入、CSV 批量导入和结果导出。

尺寸与重量口径

尺寸和商品重量先按费用变量四舍五入取到 2 位小数。体积重量及计费重四舍五入取到 3 位小数后进入分段匹配。

体积重量

公式为 长 x 宽 x 高 / 139。除小号标准和超过 150 lb 的超大件外，适用体积重量的尺寸段在计算时将宽、高最小按 2 in。

费用组合

基础 FBA 费按 <$10、$10-$50、>$50 价格段匹配，再穷举标准尺寸 SIPP 折扣或大件非 SIPP 包装费、低量库存费，并在场景最终金额中计入燃油附加费。

商品尺寸分段与计费重量

尺寸为包装后单件商品尺寸；长度加围长 = 最长边 + 2 x（次长边 + 最短边）。
